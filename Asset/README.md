# Asset

Área aislada para conceptos, exploraciones y paquetes visuales que todavía no forman parte del juego.

- Nada dentro de `Asset/` es consumido por `app/game`.
- Un concepto sólo puede migrar a `app/game/public/assets/` después de aprobación visual, normalización técnica y una tarea de integración separada.
- Conservar prompts, procedencia, estado de aprobación y límites de uso junto a cada paquete.

Paquete actual:

- `NPCs/Cafe-Cohort-v0.1/`: elenco conceptual de NPCs para una futura capa de microvida del Café Biblioteca.
