# Syka World presentation media

This folder contains repository-facing presentation media. It deliberately
keeps playable-alpha evidence separate from aspirational concept art.

## Current alpha evidence

- `syka-world-alpha-tour-v1.mp4`: freshly recorded city, Cafe Library and
  return-to-town tour from the local build.
- `syka-world-alpha-tour-preview-v1.gif`: trimmed inline preview derived from
  that recording.
- `syka-world-alpha-city-v1.png` and `syka-world-alpha-cafe-v1.png`: frames
  extracted from the same verified recording.

## Future vision

- `syka-world-future-vision-hero-v1.png`: aspirational key art generated for
  Syka World using the four approved studies under
  `research/visual-concepts/approved-direction-v1/` as visual references. It
  is not a gameplay screenshot and must remain labeled as concept art.

The future-vision image was generated with OpenAI image generation on
2026-07-21. The final prompt requested a wide, handcrafted isometric pixel-art
neighborhood at twilight, anchored by the Cafe Library, with four small
inhabitants and no UI, logos or text.
