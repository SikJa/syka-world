# Dirección visual aprobada v1

Estas imágenes son referencias conceptuales generadas para Syka World. No son sprites ni fondos listos para usar en runtime y no deben convertirse en una ciudad plana no interactiva.

## Jerarquía

1. `city-layout-a-twilight.png`: referencia principal para calidad exterior, detalle, vegetación e iluminación de atardecer.
2. `cafe-interior-library.png`: referencia principal para interiores aislados, madera, libros, chimenea, mobiliario completo y vista urbana de fondo.
3. `city-layout-b-main-street.png`: referencia de distribución libre mediante calle principal y plaza lateral.
4. `city-layout-c-green-districts.png`: referencia de microbarrios, espacios verdes y ausencia de una plaza central obligatoria.

## Reglas confirmadas

- Cámara isométrica fija; pan y zoom, sin rotación.
- Pixel art detallado y nítido; no low-poly visible.
- Ciclo de día/noche con faroles y ventanas cálidas.
- Los interiores de Whisper of the House inspiran densidad y cuidado, no se copian assets ni exteriores.
- Las ciudades pueden tener distribuciones libres creadas por el jugador.
- Los edificios aparecen amueblados al terminarse; decorar luego es opcional.
- Las imágenes conceptuales excluyen personajes para evaluar el mundo. Los avatares siguen abiertos.
